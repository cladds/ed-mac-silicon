# local/

Your personal stash. Drop custom scripts, launch wrappers, HOTAS
configurations, notes, or one-off experiments in here.

Everything inside `local/` (except this README) is ignored by git, so
nothing you put here gets pushed.

Typical uses:
- Launch wrappers with per-machine tweaks (e.g. forcing a specific GPU).
- Personal `settings.json` overrides or saved min-ed-launcher profiles.
- Scratch shell scripts you are iterating on before adding to `scripts/`.
- Notes, changelogs, or screenshots you want alongside the project.
